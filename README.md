# WordPress Backup to Google Drive

WordPress backup to Google Drive with WP-CLI. 

## Requires

* WP-CLI
* php 5.4 or later


## How to use

### Setup

```
$ curl https://.../setup.sh | bash -s
```

or

```
$ ./setup.sh
```



### Uage

```
$ curl https://.../backup.sh | bash -s
```

or

```
$ ./backup.sh
```
