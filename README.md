# WordPress Instant Backup

WordPress backup to Google Drive with WP-CLI. 

## Requires

* WP-CLI
* php 5.4 or later


## How to use

### Setup

```
$ curl https://.../setup.sh | bash -s
```

or

```
$ ./setup.sh
```



### Uage

```
$ curl https://.../run.sh | bash -s <id-dir>
```

or

```
$ ./run.sh <id-dir>
```
